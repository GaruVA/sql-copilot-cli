Brief explanation: This query will count the total orders for each customer, which can be used for various purposes such as identifying high-value customers or analyzing customer behavior.

SQL query: SELECT dbo.customers.customerid, dbo.customers.customername, COUNT(dbo.orders.orderid) AS total_orders FROM dbo.customers JOIN dbo.orders ON dbo.customers.customerid = dbo.orders.customerid GROUP BY dbo.customers.customerid, dbo.customers.customername ORDER BY total_orders DESC NULLS LAST;
